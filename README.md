# Word+One

英単語と**もう1言語**を同時に学べる単語帳Webアプリ。一緒に学ぶ言語は日本語・ドイツ語・フランス語・スペイン語などから選べます。

## 主な機能

- **検索** … 英単語を入力すると意味（英語）と、選んだ言語の訳を表示。検索した単語は自動で単語帳に追加される。画面上で「一緒に学ぶ言語」を変更可能。
- **単語帳** … 単語は20語ごとに自動で単語帳に分かれる。一覧はトグルで開閉。各単語で**削除・並び替え（↑↓）・別の単語帳へ移動**ができる。単語帳ごとに**一括で空にする**ボタンあり。
- **単語カード** … 単語帳を選んでフラッシュカード形式で復習。裏面で英訳は「全文表示」で展開可能。
- **表で確認** … 単語を表形式で表示。「単語帳」で**すべて**または**単語帳ごと**に切り替えて表示できる。

## ご利用上の注意

### 訳について
- 英語の意味は辞書API、もう1言語の訳は **Google 翻訳（非公式API）** を優先して取得し、失敗時は LibreTranslate → MyMemory の順でフォールバックしています。
- **訳は参考用です。正確でない場合があります。** 試験・論文・重要な用途では辞書や専門資料で必ず確認してください。

### 対応ブラウザ
- 動作を確認している環境：Chrome、Edge、Safari、Firefox の最新版（PC・スマホ）。
- 保存にはブラウザの localStorage を使用しているため、**シークレットモード**では終了時にデータが消えます。通常のブラウザでご利用ください。

### スマホのホーム画面に追加（推奨）
- **スマートフォンで使うときは、ホーム画面に追加しておくとアプリのように開けます。** ブックマークより起動しやすく、保存した単語も同じブラウザで開く限り引き継がれます。
- **手順：**
  - **iPhone（Safari）**  
    1. このアプリのページを Safari で開く  
    2. 画面下の **共有** ボタン（□↑）をタップ  
    3. 一覧から **「ホーム画面に追加」** を選ぶ  
    4. 名前を確認して **「追加」** をタップ  
  - **Android（Chrome）**  
    1. このアプリのページを Chrome で開く  
    2. 画面右上の **⋮（メニュー）** をタップ  
    3. **「ホーム画面に追加」** または **「アプリをインストール」** をタップ  
    4. 名前を確認して **「追加」** をタップ  
- 追加後はホーム画面のアイコンをタップするだけでアプリを開けます。

### 保存した単語について
- 単語帳のデータは**お使いの端末のブラウザ内だけ**に保存されます（サーバーには送られません）。
- **ブラウザの履歴・サイトデータの削除、アプリデータの消去、端末の初期化などを行うと、保存した単語は消え、復元できません。** 重要な単語は別途メモやエクスポートを推奨します。
- 別の端末や別のブラウザで開いた場合は、その端末にはデータがありません。

## 使用API

| 用途 | API | URL |
|------|-----|-----|
| 英語の意味・発音 | **Free Dictionary API** | https://api.dictionaryapi.dev |
| 英語→選んだ言語の訳 | **Google 翻訳**（非公式）→ LibreTranslate → MyMemory | translate.googleapis.com 等 |

- もう1言語の訳は **Google 翻訳の非公式エンドポイント** を優先しており、**利用制限（1日あたり文字数など）はありません**。接続できない場合は LibreTranslate、さらに MyMemory にフォールバックします。訳の正確性は保証されません。
- LibreTranslate を主に使いたい場合は、[Portal](https://portal.libretranslate.com/) で API キーを取得し `VITE_LIBRETRANSLATE_API_KEY` に設定するか、[セルフホスト](https://github.com/LibreTranslate/LibreTranslate)も利用できます。
