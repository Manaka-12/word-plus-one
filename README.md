# Word+One

英語ともう1言語を同時に学べる、よくばりな単語帳Webアプリ。

日本語＋ドイツ語・フランス語・スペイン語など **もう1言語を選んで学習**できます。

👉 **こちらから利用できます**
https://manaka-12.github.io/word-plus-one/

---

# 主な機能

## 🔎 検索

* 英単語を入力すると

  * 英語の意味
  * 選択した言語の訳
    を表示
* 検索した単語は **自動で単語帳に追加**
* 画面上で **「一緒に学ぶ言語」** を変更可能

---

## 📚 単語帳

* 単語は **20語ごとに自動で単語帳に分割**
* 一覧は **トグルで開閉**
* 各単語で次の操作が可能

  * 削除
  * 並び替え（↑ ↓）
  * 別の単語帳へ移動
* 単語帳ごとに **一括削除ボタン**

---

## 🃏 単語カード

* 単語帳を選択して **フラッシュカード形式で復習**
* 裏面では **英訳を「全文表示」で展開可能**

---

## 📊 表で確認

* 単語を **表形式**で表示
* 表示方法

  * すべての単語
  * 単語帳ごと

---

# ご利用上の注意

## 訳について

* 英語の意味：辞書API
* もう1言語の訳：MyMemory

訳は **参考用**です。
正確でない場合があります。

試験・論文・重要な用途では
**辞書や専門資料で必ず確認してください。**

# 対応ブラウザ

動作確認済み

* Chrome

※ PC / スマートフォン対応

シークレットモードでは終了時にデータが消えます。

---

# スマホのホーム画面に追加（推奨）

スマートフォンでは **ホーム画面に追加**すると
アプリのように使えます。

保存した単語は **同じブラウザで開く限り保持されます。**

---

## Safari

1. このアプリのページを **Safariで開く**
2. 下にある・・・から**共有**をタップ
3. 下の方にスクロールし**ホーム画面に追加** を選択
4. 名前を確認して **追加**

---

## Chrome

1. このアプリのページを **Chromeで開く**
2. 画面上にある共有ボタン（四角に上矢印）をタップ
3. もっと見るを選択し**ホーム画面に追加** を選択
4. 名前を確認して **追加**

追加後は **ホーム画面のアイコンから起動**できます。

---

# 保存した単語について

単語帳データは **端末のブラウザ内にのみ保存**されます。

以下の場合、データは消えます：

* ブラウザの履歴・サイトデータ削除
* アプリデータ削除
* 端末初期化

重要な単語は **別途メモ**をおすすめします。

# 使用API

| 用途        | API                 | URL                                 |
| --------- | ------------------- | ----------------------------------- |
| 英語の意味・発音  | Free Dictionary API | https://github.com/meetDeveloper/freeDictionaryAPI |
| 英語 → 選択言語 | MyMemory            | https://mymemory.translated.net/doc/ |

訳の正確性は保証されません。

## API利用制限

MyMemory の匿名利用制限は**1日 5,000 文字**であるため
個人の学習用途では問題になりにくい想定です。
